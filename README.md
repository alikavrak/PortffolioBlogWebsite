# Portfolio Blog Website

This project is a personal portfolio and blog web application built using PHP, HTML, CSS and JavaScript.

## Features
- User login system using PHP sessions
- Create and submit blog posts
- Dynamic blog page displaying posts
- Multi-page portfolio layout
- Frontend styling with CSS and JavaScript

## Technologies
- PHP
- HTML
- CSS
- JavaScript
- MySQL

## Purpose
This project was developed as part of a university coursework project to practice full-stack web development and database integration.
